24Data API Documentation

REST Endpoints

GET /acft-data

Aircraft data from private server. Rate limit: 75 req/min. Suggested: 20 req/min.

GET /acft-data/event

Aircraft data from event server. Shares rate limit with above. Suggested: use both for 40 req/min total.

GET /controllers

Controller positions. Rate limit: 45 req/min. Suggested: 10 req/min.

WebSocket

URL: wss://24Data.ptfs.app/wss

Max 3 connections. Remove Origin header on connect.

Events

  • ACFT_DATA – Aircraft data (main server)
  • EVENT_ACFT_DATA – Aircraft data (event server)
  • FLIGHT_PLAN – Flight plans (main server)
  • EVENT_FLIGHT_PLAN – Flight plans (event server)
  • CONTROLLERS – Controller positions

Data Structures

AircraftData

Key-value object with callsign as key. Includes heading, position, altitude, aircraft type, speed, and groundSpeed.

FlightPlan

Submitted via WebSocket. Includes aircraft, route, rules, origin, destination, etc.

Position[]

Array of controller positions. Includes queue info, airport, and role (Tower, Ground).

Usage Guidelines

  • No browser-origin requests; use a backend server.
  • Don’t exceed rate limits or retain data longer than 14 days.
  • Non-commercial use only.
  • Do not train AI models with the data.
  • API is provided as-is with no warranty.
